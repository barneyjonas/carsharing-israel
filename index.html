Service types used here

The Israeli market mixes classic carsharing and rental-like short-term access. This MVP separates them so users do not compare different models blindly.

Round-trip carsharing

Pick up and return to a defined station or parking location.

Free-floating / area-based

Pick up and return inside a defined service area, where available.

Self-service app rental

Rental-like access with app-based or low-friction pickup.

Brand short-term access

Manufacturer or network-based car access for half-day, day or longer use.

Coverage map

City-level orientation map based on the current provider database. Markers show known or approximate city coverage, not exact vehicles, stations or live availability.

If the interactive map does not load, check the city list next to it. The MVP uses external map tiles and requires an internet connection.

How to read this map

  • Markers represent cities or areas, not precise pickup points.
  • The map updates together with the provider filters below.
  • Generic entries such as “50+ cities” are kept out of the map until city-level data is verified.
